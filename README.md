# 郭辰暄的个人网站

这是一个使用 GitHub Pages 托管的个人作品展示网站。

## 网站内容

- **首页**: 个人简介和快速导航
- **关于**: 教育背景、技能和专长介绍
- **项目**: 展示的科研项目和作品
- **联系**: 邮件、GitHub 和社交媒体联系方式

## 技术栈

- HTML5
- CSS3（响应式设计）
- JavaScript（交互效果）
- GitHub Pages（免费托管）

## 快速开始

### 本地预览

1. 克隆或下载此仓库
2. 在本地文件夹中打开 `index.html`
3. 在浏览器中查看

### 在线访问

访问: `https://Guo-chenxuan.github.io`

## 自定义修改

### 修改个人信息

编辑 `index.html` 中的以下内容：
- 邮箱地址（搜索 `your-email@tsinghua.edu.cn`）
- LinkedIn 链接
- 项目内容和描述

### 修改样式

编辑 `styles.css` 文件：
- 修改 `:root` 中的 CSS 变量来改变颜色主题
- 调整字体大小和间距
- 自定义响应式布局

### 添加新项目

在 `index.html` 的 projects 部分添加新的 project-card
